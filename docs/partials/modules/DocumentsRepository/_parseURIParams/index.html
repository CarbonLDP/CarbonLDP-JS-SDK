Parse the arguments of a relative repository method detecting if has a valid URI if not, the URI will be taken from the resource provided.

Params

  • resource { } - The resource from where is executing the repository method.
  • uri { } - The possible URI to be selected or another argument.
  • args { } - All the arguments of the repository method that is been executed.

Returns

{ } - An object with the target URI detected and the rest of the arguments to be applied in the repository method.
