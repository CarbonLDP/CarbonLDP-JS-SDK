(QueryDocuments/_getMatchingDefinition/index.html)

Returns the definition from the target schema that matched the name and the URI provided. If no correct match is found, `undefined` will be returned.

Params

  • generalSchema { } - The schema to use in the resolution of relative URI in the matched definition.
  • targetSchema { } - The schema where to look for the property's definition.
  • propertyName { } - The name of the property to look for.
  • propertyURI { } - The optional URI that the property has to have for have a true matching.
